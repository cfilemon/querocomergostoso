RESTful API documentation

Disclaimer: this API is under development. Although all the routes listed below are up and working, they may change their behaviour or just disappear with no warning. ¯\_(ツ)_/¯

GET /restaurant

Returns all restaurants

POST /restaurant

Creates a new restaurant

GET /restaurant/{id}

Returns a restaurant whose id is given

DELETE /restaurant/{id}

Deletes a restaurant whose id is given

PATCH /restaurant/{id}

Patches a restaurant whose id is given. Note: online replace operations are allowed!

GET /restaurant/{id}/menu

Returns the menu of the restaurant whose id is given

POST /restaurant/{id}/menu

Creates a new menu for that restaurant.Note: If the restaurant already has one, it'll be replaced.
