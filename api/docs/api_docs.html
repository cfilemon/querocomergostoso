RESTful API documentation

last edit 30-07 @ 0h14 by clenimar

Disclaimer: this API is under development. Although all the routes listed below are up and working, they may change their behaviour or just disappear with no warning. ¯\_(ツ)_/¯

GET /restaurant

Returns a collection of all the restaurants.

POST /restaurant

Creates a new restaurant.

GET /restaurant/:id

Returns the restaurant specified by the id parameter.

DELETE /restaurant/:id

Deletes the restaurant specified by the id parameter.

PATCH /restaurant/:id

Patches the restaurant specified by the id parameter. Note: only replace operations are allowed!

GET /restaurant/:id/menu

Returns the menu of the restaurant specified by the id paramemeter.

POST /restaurant/:id/menu

Creates a new item in the menu of the restaurant specified by the id parameter.
